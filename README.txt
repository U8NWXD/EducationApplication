INSTALLATION INSTRUCTIONS (FOR MAC OS X)

For now, this program cannot be run as a normal stand-alone
application. Instead, the compiled object code must be run from the
command line on a computer with Java installed.

System Requirements:
    Sufficient memory, processing power, and disk space
        Very little is required
    Java 8
        Up-to-date version recommended
    Mac OS 10.11.6
        Other Mac OS X versions may work but are untested
        Other operating systems may have incompatible file systems

Integrity Verification and Operating System Warnings:

Whatever form of the program you receive should be signed in some
way by the author's OpenPGP key (fingerprint available in contact
information and public key included with program). You should verify
this signature and the key before running the program to ensure the
program has not been tampered with or corrupted in transit. However,
because the program has not been signed by keys registered with
any operating system manufacturer, the program may be flagged as
untrusted. Follow your operating system manufacturer's instructions
for bypassing these restrictions only after you have verified the
program's source and integrity in another way.

To install the program, move it to a convenient directory. You will
have to navigate to this directory to launch the program each time.
Then, open a terminal window and navigate to the following directory:
    EducationApplicationCode/out/production
Then, to initialize the file system directory structure, run (all on
one line with a space between the two lines below:
    java -cp EducationApplication
    com.icloud.cs_temporary.EducationApplication.Initializer
